categories: - Programming/By Language/Java - Programming/By Language/Scheme - Programming/By Project/Wombat IDE comments: true date: 2012-02-29 04:55:40 layout: post title: Wombat IDE - Updated draw-image --- After a long while, I've actually done some work on the frame displayed with `draw-image`. Now rather than just showing the basic image, there's a bit of extra information available. @embed{new-draw-image.png} The new dialog will show the size of the image, the current size it is being drawn at (which will automatically scale with the size of the dialog), and the color being displayed under the cursor. In addition, the initial size of the dialog is based on the size of the image, automatically scaling images that would normally be either too small or too large to sensibly display. As a small side note, the predefined colors from the @crosslink["c211-tree-and-image-libraries"]{old image API} are now defined again. Edit: The newest C211 Image API can be found here. If you would like to download the new version of Wombat, you get get the launcher here. You should have at least version 2.58.2.
