categories: - Programming/By Language/Java - Programming/By Language/Scheme - Programming/By Project/Wombat IDE comments: true date: 2012-05-03 04:55:34 layout: post title: Wombat IDE - Vector of vector functions --- Added two functions to the matrix library:
  • `(vov->matrix vov)` - converts a vector of vectors to a matrix
  • `(matrix->vov m)` - converts a matrix to a vector of vectors
Quick and simple, but highly useful for testing and display purposes.
