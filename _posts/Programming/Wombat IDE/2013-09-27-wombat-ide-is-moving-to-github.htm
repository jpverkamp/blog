To that end, we've moved from Google Code: wombat-ide to GitHub: jpverkamp/wombat-ide. The first part of the change was the source code and version control history. This actually went relatively straight forward. All I had to do was clone the code to a local machine, pull down all of the branches, replace the 'origin' repository, and push.
